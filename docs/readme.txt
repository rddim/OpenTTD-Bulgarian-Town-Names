1. Description
-----------------

This NewGRF provide existings Bulgarian villages,
towns and cities names for OpenTTD.

2. Requirements
-----------------

- OpenTTD http://www.openttd.org
- OpenTTD Portable http://portableapps.com/apps/games/openttd_portable

3. Installation
-----------------

Please use [BaNaNaS] http://bananas.openttd.org/en/newgrf for installation
or the options for online content in the OpenTTD

4. Configuration
-----------------

Execute OpenTTD.
Choose [NewGRF Settings] and add <Bulgarian Town and Village Names vX.X>.
Then go back and choose [Game Options] and select one of the options from [Town names] selection menu.
